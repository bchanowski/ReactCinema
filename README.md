# React Cinema Project

This project is a frontend-only cinema web application built with React and TypeScript. The app fetches live movie data from an external API and provides an intuitive interface. Used technologies are: 
- React TS
- TailwindCSS
- React-DOM
- Vite
- React-Router
- Zustand
- React-Query
- Lucide
